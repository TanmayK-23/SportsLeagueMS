# 🏟️ Sports League Management System (SLMS)

A web application built with **Flask** and **MySQL** to manage sports data with a modern, responsive interface.

## ✨ Features

- **Player Database**: Browse cricket and football players with real-time search and filtering.
- **Dynamic Stats**: Live row counters and role-based badges.
- **Easy Registration**: Simple registration flow with sport-specific position selection.
- **Readable Data**: Formatted dates and clean data presentation.

## 🚀 Tech Stack

- **Backend**: Python / Flask
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript

## 🛠️ Installation & Setup

### 1. Setup Environment
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Database Configuration
1. Create the database: `CREATE DATABASE SportsLeagueDB;`
2. Update connection details in `app.py`.

### 3. Run App
```bash
python app.py
```
